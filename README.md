# Ecommerce_data_analytics
## E-Commerce Sales &amp; Customer Analytics Project 
### Project Overview

This project demonstrates end-to-end data analytics and visualization using Python and Excel on an e-commerce dataset. The work focuses on transforming raw transactional data into meaningful insights through data cleaning, exploratory analysis, visualization, KPI computation, and dashboard creation. The project reflects practical data analyst workflows commonly used in real-world business scenarios.

### Objectives

The objective of this project is to clean and preprocess e-commerce transaction data, explore patterns and trends through analysis, visualize insights using Python libraries, and present summarized results through interactive Excel dashboards to support data-driven decision-making.

### Dataset Description

      Dataset Type: E-commerce transactional data

      File Format: CSV

      Number of Records: ~5,000

      Source: Kaggle

### Columns:

    Order details: order_id, order_date

    Customer details: customer_id, age, gender, city

    Product details: product_category, unit_price, quantity

    Transaction details: discount_amount, total_amount, payment_method

    Engagement & satisfaction: session_duration_minutes, pages_viewed, delivery_time_days, customer_rating

    Customer type: is_returning_customer

### Tools & Technologies Used

Python (Pandas, NumPy), Matplotlib, Microsoft Excel (Pivot Tables, Charts, Slicers), Google Colab, Kaggle.
